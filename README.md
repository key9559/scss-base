# scss-base

##광범위하게 사용가능한 scss base 제작
###base
```
  -base
    =reset custom...
  -helper
    =common class
  -typo
    =font-face...
```
###component
```
  -component styles
    =card, list, input, animation...
```
###layout
```
  -layout styles
    haeder, footer, l-~...
```
###page
```
  -page contant styles
    =main, cs, auth, enroll, mypage...
```
###utility
```
  -variable
  -mixin
  -function
```
###vendor
```
  -output styles
    =reset, normalise, slick, swiper...
```
